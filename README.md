# RATBV App 🚌

An unofficial mobile app for the RATBV public transport network in Brașov, Romania. Browse bus and trolleybus routes, view stop lists, switch directions, and check timetables — all from your phone, offline.

Built with Expo and React Native using official GTFS data from RATBV S.A.

## Features

- 📋 Browse all bus and trolleybus routes with official route colors
- 🛑 View the full stop list for any route in order
- 🔄 Switch between outbound and return directions
- 🕐 View timetables for a specific stop on a specific route
- 📞 Contact info and links to the official RATBV website
- 📴 Works fully offline — no internet connection required
